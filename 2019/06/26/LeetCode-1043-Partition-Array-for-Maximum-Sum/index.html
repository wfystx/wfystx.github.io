1043. Partition Array for Maximum Sum

  • Topic:

    dynamic programming

  • Question:

    Given an integer array A, you partition the array into (contiguous) subarrays of length at most K. After partitioning, each subarray has their values changed to become the maximum value of that subarray.

    Return the largest sum of the given array after partitioning.

    Example 1:

    Note:

    1. 1 <= K <= A.length <= 500
    2. 0 <= A[i] <= 10^6

  • Idea:

    The idea of this problem is to creat a vector to store the current largest sum value, which is: dp[i] = sum of A[0] ~ A[i]. So we take every number in the input array, find the largest number in subarray of k numbers before current number, we can use this number k times.
