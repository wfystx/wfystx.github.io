46. Permutations

  • Topic:

    backtracking

  • Question:

    Given a collection of distinct integers, return all possible permutations.

    Example:

  • Idea:

    In this backtracking problem, we need to figure out all possibilities, the point is: 1. no duplicates. 2. each number could be in each position.

    We can use the tradition recursive method of backtracking to push a number into sub array and start at the very beginning of input because every number could be in every position. To handle duplicates, we use a vector of bool to mark if the number is already used before.

    Another solution is not to push, but swap two numbers every time, and start at the current postion +1 in next recursion. When the pos gets the end of the input, we are done.
