Learn C++ -- atoi() and stoi()

Posted on 2019-05-26 | Edited on 2019-05-31 | In coding | Comments:

The same points are:

  • Both atoi() and stoi() are used for convert a number string to int.
  • Their library is #include <cstring>.

The differences are talked below:

  • Parameter for atoi() is const char*. So we need to use c_str() to convert a string to const char* before use. But the parameter for stoi() is const string*, so we don’t need to convert it.
  • stoi() function will do a range check before use, if the parameter is not in range [-2147483648, 2147483647], it throws an exception, a runtime error. However, atoi() doesn’t do that. If the parameter is out of range of int, it causes undefined behavior.
