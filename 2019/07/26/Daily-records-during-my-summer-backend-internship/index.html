Daily records during my summer backend internship

Posted on 2019-07-26 | Comments:

It has been 3 weeks from I came to Stars Shining Tech Co. Ltd to work as the backend intern. I finally realized that I should write down something to record my experience here, what I did, what I learned. Hope I will do this record frequently in the future~

The first 3 weeks

  • Learned the basics of PHP programming language and some common functions. PHP is a beautiful language which runs fast, and the learning curve is also fast. Because I have the fundamental of other programming languages, so this was not so hard.
  • Reviewed sql sentences and can write them faster~ With the help of High performance Mysql, I got a deeper view of this fantastic database, including its storage engine, how it stores. Not to mention the most important thing: index. What is index and how it works.
  • By reading Git Pro, I got a better understanding of git. In the past, I only knew some basic commands like add, commit, push. But now, in some level, I became much better. I learned how branches work, when should I use them, how to use them in a project of company, how to work with others at the same time, and also how to use the commands to make sure I’m not making any mistakes.
  • I also got a deeper understanding about MVC framework since I started to read the backend code of our company. I searched on internet to know what is MVC framework, how it is formed, how it works. And by reading the source code of Phalcon framework, I learned how it starts. It’s really helpful for me to understand the whole project. I can code more efficiently in the future, and when I got bugs, I know how to find it step by step.
  • I have to mention some words I just had a vague picture when studied at school: HTTP protocal, API, Request, Response…. They are so important for us but I didn’t realized until now.
  • For now, I’m able to code some apis or fix some bugs by myself. I’m getting to know the logic of our project better, and getting familiar with our code. I’m sure I will do it great next!
